    • Configuration

    As stated in the Getting Started section, all of the The configuration options provided by the jsdoc configuration are retained. jsdocp only adds addtional configuration options to the existing jsdoc configuration opts.

    Configuration Options

    jsdocp offers many addon options in additon to the jsdoc configuration. A full listing of all the jsdocp addon configuration options can be found under publicize. All paths defined in the configuration are relative to the module the docs are being generated for.
